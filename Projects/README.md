# Tutoring Web

## 📖 Description
A web application that connects tutors with learners. Users can switch between tutor and learner modes and interact based on their needs.

## 🚀 Technologies Used
- HTML
- CSS
- JavaScript
- Node.js

## ⚙️ Installation
1. Clone the repo:
   git clone https://github.com/Kaizerklaas/tutoring-web

2. Open the project folder

3. Run in browser

## ▶️ Usage
- Select tutor or learner mode
- Search for subjects
- Connect with users
