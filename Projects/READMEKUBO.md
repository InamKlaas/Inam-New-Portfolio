# Kubo Online Store

## 📖 Description
An e-commerce website for buying branded clothing. Users can browse products, add to cart, and place orders.

## 🚀 Technologies Used
- HTML
- CSS
- JavaScript

## ⚙️ Installation
1. Clone the repo:
   git clone https://github.com/Kaizerklaas/kubo-store

2. Open index.html

## ▶️ Usage
- Browse products
- Add items to cart
- Checkout
